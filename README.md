# 5 Minute Journal

> This project is no longer being maintained

A Journal App taken from [5 Minute Journal](https://www.intelligentchange.com/products/the-five-minute-journal).

# Build
```bash
cd packages/webapp/
npm run build
```

# Run
```bash
cd packages/server/
npm run start
```

# Lint
```bash
# Lint server
cd package/server/
npm run lint:fix
npm run format

# Lint webapp
cd package/webapp/
npm run lint:fix
npm run format
```
